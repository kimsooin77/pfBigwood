# Big Wood portfolio

- react로 작성된 웹사이트입니다.
- 메인페이지를 포함하여 department, community, gallery, youtube, location, join 총 7페이지로 구성되었습니다.
- 반응형처리로 desktop, tablet, mobile 기기에서도 이용 가능하도록 제작되었습니다.
- community와 youtube 컴포넌트는 리덕스를 적용하여 효육적으로 상태관리할 수 있도록 구현하였습니다.
